PROJECTS

  1. Medical Operating Room Vitals & Stay Predictions

    Python, Exploratory Data Analysis, Predictive Models, Statistics

    Developed a predictive analytics solution for hospital resource management, focusing on patient length of stay predictions. The project reduced data size by 90% and achieved 82% accuracy in forecasting patient outcomes using Gradient Boosted Trees with Bayesian optimization.

    • Achieved a 90% reduction in data size through data cleaning, encoding, and feature selection.
    • Explored healthcare patterns, including gender and age-specific trends in diseases, post-operative complications, and hospital length of stay using visualization and analytics libraries like Plotly, Matplotlib, and Seaborn.

    SQL Server, Azure Blob Storage, Azure Data Factory, PySpark, Databricks, Power BI, KPIs

    Built ETL pipelines to transfer data from SQL Server to Azure Blob Storage, then fetched it into Databricks for cleaning with PySpark. Stored the cleaned data back in Azure Blob Storage and analyzed it in Power BI, developing dashboard with KPIs to track pizza sales trends over various timeframes.

    • Developed ETL pipelines in Azure Data Factory to transfer data from SQL Server to Azure Blob Storage, cleaned the data in Databricks using PySpark, and stored the results back in Blob Storage.
    • Analyzed the cleaned data in Power BI and created dashboard with KPIs to track pizza sales trends across various timeframes.
